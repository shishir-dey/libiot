HTTP client implementation.

Provides a simple HTTP/1.1 client suitable for embedded systems, supporting GET and POST methods with custom headers. HTTP/1.1 protocol implementation for embedded systems.

This module provides a lightweight HTTP client implementation designed specifically for embedded systems and no_std environments. It focuses on simplicity, predictable memory usage, and compatibility with resource-constrained devices.

ยงFeatures

  • HTTP/1.1 protocol compliance
  • Synchronous request/response model
  • Fixed-size buffers for predictable memory usage
  • Support for custom headers
  • GET and POST method support
  • Connection reuse capability
