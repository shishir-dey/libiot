Expand description

RAM-based storage operations.

RAM storage provides the fastest access but is volatile. This trait provides operations specific to RAM-based storage including direct memory access.

fn clear(&mut self) -> Result<(), <Self as ReadStorage>::Error>

Clear the entire storage area.

Sets all bytes in the storage to zero. This is only meaningful for volatile storage as non-volatile storage would lose this state on power loss.

§Returns
  • Ok(()) - Storage cleared successfully
  • Err(error) - Clear operation failed
