Expand description

SD/MMC card specific operations.

SD and MMC cards require special initialization sequences and have specific status information that can be useful for applications.

fn set_block_length( &mut self, len: u32, ) -> Result<(), <Self as ReadStorage>::Error>

Set the block length for SDSC cards.

Standard Capacity (SDSC) cards support configurable block lengths. High Capacity (SDHC/SDXC) cards always use 512-byte blocks.

§Arguments
  • len - Block length in bytes (typically 512)
§Returns
  • Ok(()) - Block length set successfully
